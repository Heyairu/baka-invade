侵略性詞彙是什麼？

經過中國的言論審查、共產思想而產生的詞彙，例如：中國提倡不說英文而將「RUN」講為「潤」；避免言論審查而將「和諧」講為「河蟹」。

語言代表一個國家的文化與思想，若長時間使用這些詞彙，可能會在無形之中影響自己的思考方式。
